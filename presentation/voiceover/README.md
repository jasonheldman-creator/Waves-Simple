# WAVES Intelligence™ Voiceover Generation

This directory contains the ElevenLabs voiceover generation pipeline for the WAVES Executive Briefing.

## Quick Start

```bash
# 1. Set your ElevenLabs API key
export ELEVENLABS_API_KEY="your_api_key_here"

# 2. Install dependencies
pip install elevenlabs pydub

# 3. Generate audio files
python generate_audio.py
```

## Files

- **elevenlabs_config.json** - Voice configuration (Adam voice, professional settings)
- **generate_audio.py** - Audio generation script
- **NARRATION_MANIFEST.json** - Generated manifest with durations and mappings
- **slide_01.mp3 through slide_13.mp3** - Generated audio files

## Configuration

The `elevenlabs_config.json` file contains:
- Voice ID and model selection
- Voice settings (stability, similarity, style)
- Output format (MP3 at 44.1kHz, 128kbps)
- SSML pause support

## Script Features

1. **Reads narration text** from `/presentation/narration/`
2. **Generates high-quality audio** using ElevenLabs API
3. **Creates placeholders** if API key is missing (silent audio or marker files)
4. **Validates durations** against 7-9 minute target
5. **Generates manifest** for PowerPoint integration

## Output

The script generates:
- 13 MP3 audio files (one per slide)
- NARRATION_MANIFEST.json with complete metadata
- Duration analysis and warnings

## Requirements

```bash

```

## Troubleshooting

See the main README.md in `/presentation/` for detailed troubleshooting instructions.

### Quick Fixes

**No API key:**
- Script automatically creates silent placeholders
- Set `ELEVENLABS_API_KEY` environment variable for real audio

**ffmpeg not found:**
- Install: `brew install ffmpeg` (macOS) or `apt-get install ffmpeg` (Linux)
- Script will create marker files if ffmpeg unavailable

**Rate limits:**
- Script includes automatic rate limiting (0.5s between calls)
- Check quota at https://elevenlabs.io/
